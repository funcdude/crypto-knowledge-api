🚀 QUICK START (ONCE CREDENTIALS PROVIDED)

  Step 1: Update credentials
  $ cd ~/crypto-knowledge-api
  $ echo "PINECONE_API_KEY=pc_your_key_here" >> .env

  Step 2: Rebuild and start
  $ docker compose build api
  $ docker compose down && docker compose up -d db redis api

  Step 3: Wait for startup (watch logs)
  $ docker logs -f crypto-knowledge-api-api-1

  Step 4: Validate Pinecone connection
  $ ./scripts/test-pinecone.sh

  Step 5: Test search endpoint
  $ curl -X POST http://localhost:8000/api/v1/search \
    -H "Content-Type: application/json" \
    -d '{"query": "What is Bitcoin?", "tier": "explanation"}'
