  1. Get API Key (1 min)
     → https://app.pinecone.io/api-keys

  2. Configure (2 min)
     → export PINECONE_API_KEY="pc_..."
     → Add to .env

  3. Rebuild & Start (5 min)
     → docker compose build api
     → docker compose down && docker compose up -d db redis api

     → ./scripts/verify-pinecone.sh

  5. Test Search (10 min)
